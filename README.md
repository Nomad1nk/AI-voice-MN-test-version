# AI Voice Operator (Mongolian Test Version)

This project is a voice-enabled AI receptionist designed for "Sky Dental Clinic". It uses OpenAI's GPT-4o for intelligence and Realtime API for speech capabilities, specifically tailored to understand and speak Mongolian.

## Features
- **Mongolian Language Support**: Customized system prompts and voice settings for Mongolian interaction.
- **Real-time Voice Interaction**: Uses WebRTC for low-latency voice conversations.
- **Web Interface**: A simple frontend to interact with the AI agent.

## Setup
1.  Install dependencies: `pip install -r requirements.txt`
2.  Set up environment variables in `.env` (OPENAI_API_KEY).
3.  Run the server: `python main.py`
